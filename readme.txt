== Description ==

Este plugin te permitira tener un boton en la parte lateral de tu web el cual muestra un reproductor de video de musica. Aprovecha la API de GOODFIDELITY para administrar contenido relevante.

= Features =

- En un boton tienes una web site.
- Mira y escucha videos musicales.
- Selecciona tus artistas favoritos.
- No modifica ni altera tu tema.
- No ocupa espacio.
- Mejora la experiencia de tus usuarios.
- Utiliza Ajax para cargar la pagina

== Installation ==

Play Video of Song is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. Copy Folder from the downloaded and extracted file.

2. Paste it in wp-Content/plugins folder on your Wordpress Installation 

3. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated.
